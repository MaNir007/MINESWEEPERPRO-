 1. EKONOMIJA I PERKOVI (CRNO TRŽIŠTE)
=============================================================================
Napredovanje kroz igru nagrađuje vas lokalnom valutom: Coins (novčićima).

💰 KAKO ZARADITI?
- Svaka očišćena mina i pobjeda u igri donosi resurse.
- Veća težina polja = veća fiksna zarada.

🛠️ CRNO TRŽIŠTE (SHOP):
Na glavnom meniju nudi vam se nabavka taktičke opreme:
  [📡] RADAR (50 C) : Skenira 3x3 područje. Prikazuje mine na 2.5 sekunde.
  [🛡️] DEFUSE KIT (100 C) : Spašava život jednom pri naletu na minu.


=============================================================================
 2. STANDARDNI MODI IGRE
=============================================================================

🎯 TRENING (Training)
Klasičan Minesweeper. Razine: Easy (6x6), Medium (10x10), Hard (15x15), Impossible (20x20).
Custom mod dopušta kreiranje mapa do 30x30.

🏆 KARIJERA (Career Mode)
Stalna progresija. Svaka pobjeda povećava vaš Level. Što je veći level, veća je 
mapa i broj mina, ali i prestiž na globalnoj ljestvici.

🗓️ DNEVNI IZAZOV (Daily Challenge)
Fiksni "seed" generiran na osnovu današnjeg datuma. Svi igrači igraju istu mapu.


=============================================================================
 3. SPECIJALNE OPERACIJE (MINI IGRE)
=============================================================================

⚡ MINE DASH (Trka kroz mine)
Brzinski mod. Morate uloviti točna polja u nizu dok vrijeme ističe. Točan klik = +2s.

💎 TREASURE HUNTER (Memorija)
Mine se vide na početku 2 sekunde, a onda nestaju. Morate otvoriti sva prazna
polja isključivo po sjećanju.

🧩 CHAIN REACTION (Bez otvaranja)
Ploča je već "otvorena" brojevima. Vaš zadatak je postaviti zastavice bez greške.
Svaka kriva zastavica oduzima vam energiju (HP).

🧘 ZEN VRT (Relax mod)
Nema smrti. Svaki klik na minu samo je označi kvačicom. Za opušteno igranje.

🌫️ MAGLA (Fog of War)
CIjela mapa je crna (mrak). Vidite samo radius oko zadnjeg polja koje ste kliknuli.


=============================================================================
 4. BRZI PREGLED KONTROLA
=============================================================================
Lijevi klik: Otkrivanje polja (reveal).
Desni klik: Postavljanje zastavice (flag).
Chording: Klik na broj (ako je oko njega već postavljen ispravan broj zastavica) 
automatski otvara sva ostala susjedna polja.


=============================================================================
 5. TEHNIČKA ARHITEKTURA (Pregled datoteka)
=============================================================================

[ ROOT ]
• index.html : Glavna struktura aplikacije. Sadrži kontejnere za modale, grid za 
               igru i upravljanje svim vizualnim elementima (HUB).
• api.php    : Backend kontroler (PHP). Upravlja SQL bazom, SQL transakcijama 
               (spremanje coinsa, levela) i vraća Leaderboard podatke u JSON formatu.
• baza.sql   : SQL shema tablica za korisnike, njihove tokene i postignuća.

[ /CSS ]
• base.css   : Globalni stilovi (layout, fontovi, glassmorphism kartice).
• friv.css   : Stilovi za 3D tile izbornik (Friv stil). Gradijenti i animacije tilesa.
• game.css   : Dizajn same igračke ploče (ćelija, boja brojeva, efekti eksplozije).
• themes.css : Skup klasa (.theme-neon, .theme-ocean, itd.) koje mijenjaju izgled 
               cijele aplikacije i boja pločica jednim klikom.

[ /JS ]
• globals.js : Pohrana globalnih varijabli (level, coins, player name, grid array).
• game.js    : Logika igre. Algoritmi za postavljanje mina, BFS algoritam za 
               automatsko otvaranje praznih polja (flood fill) i mehanike mini-igara.
• ui.js      : Upravlja DOM-om. Otvaranje/zatvaranje modala, učitavanje tema sa 
               localStorage, animacije (konfeti, notifikacije).
• svg-loader.js: Sustav za rendering vektorskih ikona. Sadrži veliki objekt 'Icons' s 
               sirovim SVG kodom koji se ubrizgava u 'data-svg' elemente.
• timer.js   : Upravlja vremenom partije i specifičnim timerima u mini-igrama.
• api.js     : JS klijent za komunikaciju s api.php (fetch() pozivi).
• cheats.js  : Developer konzola za testiranje (xray(), godmode(), addCoins(n)).


=============================================================================
 6. FUNKCIJSKI VODIČ (Glavne funkcije)
=============================================================================

[ game.js ]
• initGame() : Resetira stanje i kreira Grid matricu (dvodimenzionalni niz objekata).
• placeMines() : Nasumično razmješta mine pazeći da se mina ne pojavi na prvom kliku.
• revealCell(x, y) : Glavna funkcija koja otvara polje. Ako je polje prazno (0), 
                     rekurzivno poziva otvaranje susjeda.
• checkWin() : Provjerava je li broj zatvorenih polja jednak broju mina.

[ ui.js ]
• applyTheme() : Dohvaća ime teme iz localStorage i primjenjuje odgovarajuću 
                 klasu na <body> element (npr. body.theme-matrix).
• showOverlay(title, text) : Prikazuje pobjednički/gubitnički modal s ikonom.
• openThemeModal() : Prikazuje prozor za izbor dizajna (Theming Terminal).

[ svg-loader.js ]
• injectStaticSVGs() : Prolazi kroz cijeli HTML i u svaki element koji ima `data-svg="ime"`
                       ubrizgava odgovarajući SVG kod iz memorije.


=============================================================================
 7. Q&A - ARHITEKTURALNA PITANJA I ODGOVORI
=============================================================================

P: Kako se vrši odabir game moda u Friv izborniku?
O: Svaka pločica u Friv gridu ima onclick događaj koji poziva `startGame('ImeModa')`. 
   Funkcija `startGame` u game.js uzima taj parametar, postavlja konfiguraciju 
   (veličinu mape i broj mina) te pokreće `initGame()`.

P: Na koji način radi sustav učitavanja tema bez ponovnog učitavanja stranice?
O: Sustav koristi CSS specifičnost (Specificity). Sve teme su definirane u 
   themes.css kao podklase body elementa (npr. `body.theme-neon .card`). Kada u 
   JS-u pomoću `applyTheme()` promijenimo klasu na <body> elementu, CSS pravila 
   se automatski ažuriraju za sve elemente na ekranu.

P: Kako radi SVG sustav (zašto ne koristite .svg slike)?
O: Korištenje inline SVG koda (preko svg-loader.js) omogućuje nam tri stvari: 
   1. Nema HTTP requestova za svaku sliku (brže učitavanje). 
   2. Možemo animirati dijelove SVG-a direktno kroz CSS. 
   3. Dinamički mijenjamo boje (`fill="currentColor"`) što nam omogućuje da 
      ikone mijenjaju stil ovisno o odabranoj temi.

P: Kako je osigurana sigurnost spremanja rezultata i coinsa?
O: Rezultati se ne spremaju lokalno na klijentu gdje ih korisnik može lako izmijeniti. 
   Umjesto toga, js poziva `api.js` (fetch), koji šalje podatke PHP backendu. 
   Backend vrši provjeru i tek onda vrši upis u SQL bazu. Kod Daily Challenge-a, 
   provjerava se i integritet "seeda".

P: Kako radi flood-fill (automatsko otvaranje praznih polja)?
O: Koristi se rekurzivni pristup u funkciji `revealCell`. Ako polje koje ste kliknuli 
   ima 0 susjednih mina, algoritam automatski poziva `revealCell` za svih 8 
   susjednih polja, sve dok ne dođe do polja s brojem.

P: Kako radi Daily Challenge (Dnevni Izazov) sustav?
O: Koristimo Mulberry32 algoritam koji je deterministički PRNG (Pseudo-Random 
   Number Generator). Kao ulazni parametar (seed) dobiva trenutni datum 
   (YYYYMMDD). To garantira da će funkcija za generiranje mina na svim 
   računalima svijeta danas vratiti identičan raspored mina.
